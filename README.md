## 功能列表

- **学生信息管理**：
  - 添加学生信息（姓名、学号、各科成绩）。
  - 删除学生信息。
  - 修改学生信息。
  - 查询学生信息。
- **成绩统计与排名**：
  - 计算每个学生的平均成绩。
  - 按平均成绩对学生进行排名。
  - 显示所有学生的排名信息。
- **班级成绩统计**：
  - 统计并显示班级的最高分、最低分和平均分。
