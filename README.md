# 学生管理系统

出于学习目的写的一个基于c的简易学生管理系统，支持班级学生信息的增删改查、成绩排名统计管理等功能，目前这个系统还十分简陋，之后有可能会进一步优化该管理系统

## 功能列表
- 学生信息管理：添加、删除、修改、查询学生信息。
- 成绩管理：录入、修改、查询学生成绩。
- 统计功能：按学生的平均成绩进行排名，统计班级平均分、最高分以及最低分
